# 📚 Enhanced Library Management System (Python CLI)

A command-line based Library Management System built with Python.  
This project supports **multiple users**, **book categories**, **book availability tracking**, and **data persistence** using JSON files.

---

## 🚀 Features

- ✅ Add books with category and number of copies
- 📖 Display all available books with category and availability count
- 👤 Borrow and return books as different users
- 💾 Data persistence (all book and user data saved in `library_data.json`)

---

## 🛠️ Tech Stack

- Python 3
- JSON for file-based data storage
